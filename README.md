# Painel Validador de Senhas

Projeto em Python com interface gráfica em Tkinter para criação, edição, exclusão, listagem e validação de senhas.

## Funcionalidades
- Criar senha
- Editar senha
- Deletar senha
- Listar senhas mascaradas
- Validar senha com base em requisitos
- Classificar senha como fraca, média ou forte

## Tecnologias usadas
- Python
- Tkinter

## Como executar
```bash
python gui.py
```
## Objetivo
Este projeto foi desenvolvido com o objetivo de praticar:

- lógica de programação
- manipulação de listas
- funções em Python
- validação de dados
- criação de interface gráfica com Tkinter

## Observação
As senhas são mascaradas visualmente na interface, tornando a exibição mais segura durante o uso da aplicação.
